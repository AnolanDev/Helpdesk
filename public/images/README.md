# Directorio de Imágenes

Este directorio contiene las imágenes públicas de la aplicación.

## Logo

**Archivo:** `logo.jpg`

El logo se muestra en:
- Header principal (AppLayout) - 40x40px
- Página de login (GuestLayout - versión desktop y móvil) - 64x64px
- Página de bienvenida (Welcome) - 48x48px

**Especificaciones recomendadas:**
- Formato: JPG, PNG o SVG
- Tamaño: 512x512px (o proporcional cuadrado)
- Peso: < 500KB para optimizar carga
- El logo debe ser cuadrado para mejor visualización

## Agregar el logo

Coloca tu archivo `logo.jpg` en este directorio:
```
public/images/logo.jpg
```

El logo se renderizará automáticamente en toda la aplicación.
